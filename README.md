# Insurance Cost Prediction (Machine Learning)

This project predicts **medical insurance charges** based on customer details such as age, BMI, smoking status, number of children, and region using **Machine Learning**.

The goal of this project is to analyze the dataset, visualize patterns, and build a model that can predict insurance charges.

---

##  Project Overview

Medical insurance charges depend on multiple factors. In this project, we:
- Perform data cleaning and preprocessing
- Visualize relationships between features and insurance charges
- Encode categorical features
- Train machine learning models
- Evaluate model performance

---

##  Dataset Information

The dataset contains the following features:

| Column Name | Description |
|------------|-------------|
| age | Age of the person |
| sex | Gender (male/female) |
| bmi | Body Mass Index |
| children | Number of children |
| smoker | Smoker or non-smoker |
| region | Region of the person |
| charges | Insurance charges (Target column) |

##  Machine Learning Workflow

1. Load dataset
2. Data exploration (EDA)
3. Handle missing values (if any)
4. Encode categorical variables
5. Split dataset into training and testing set
6. Train model (Regression model)
7. Evaluate performance using metrics (R2 score, MAE, MSE)
